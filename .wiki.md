# Project Summary
The project is a web application for "Suleiman Al-Huwaiti," a customs clearance and transit services company. It provides efficient logistics solutions, including customs clearance, shipping, and transit services, primarily for the Saudi Arabian market. The application enhances user experience through a modern interface, optimized performance, and vibrant design elements, with improved color schemes for a more professional look.

# Project Module Description
The application is organized into several functional modules:
- **User Interface**: Contains components for navigation, service details, a blog, and contact information.
- **SEO Optimization**: Utilizes structured data and meta tags to improve search engine visibility.
- **Responsive Design**: Ensures the application is usable across different devices with a mobile-friendly layout.

# File Description Inventory
- **README.md**: Project documentation and setup instructions.
- **components.json**: Configuration for UI components.
- **eslint.config.js**: ESLint configuration for maintaining code quality.
- **index.html**: Main HTML file with the structure and SEO meta tags.
- **package.json**: Lists project dependencies and scripts.
- **public/**: Contains static assets like images, favicons, and SEO-related files (robots.txt, sitemap.xml).
- **src/**: Source code for the application, including styles, components, hooks, and the main entry point.
- **src/sections/Hero.tsx**: Implements a dynamic hero section with a rotating image display, vibrant colors, enhanced user interaction features, and improved color schemes for better visibility. Recent updates include improved button functionality, text clarity, and overall user experience enhancements.

# Technology Stack
- **Frontend**: React (TypeScript), Vite, Tailwind CSS
- **SEO**: JSON-LD for structured data, Open Graph, and Twitter Card meta tags
- **Build Tools**: pnpm for package management

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Lint the code:
   ```bash
   pnpm run lint
   ```
3. Build the project:
   ```bash
   pnpm run build
   ```
